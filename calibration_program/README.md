# WEB3 stereo calibration

Запускайте из корня проекта файл `CALIBRATE_STEREO.bat`.

Он освобождает камеры, сохраняет резервную копию действующей калибровки в
`calibration_backups/` и запускает `full_calibrate_v2.py`. Новый файл
`stereo_calibration.yaml` записывается непосредственно в корень проекта,
откуда его читает сервер.

Доска: 7 × 6 внутренних углов (8 × 7 клеток), размер клетки 22.8125 мм.

Вычисление использует бюджет из 16 логических процессоров: левая и правая
монокалибровки считаются одновременно как два задания по 8 потоков, после чего
стереокалибровка получает все 16 потоков. Результат последнего прохода удаления
выбросов переиспользуется, поэтому одинаковая монокалибровка повторно не считается.

- `C` — снять пару кадров;
- `Space` — рассчитать и сохранить после 20 разных ракурсов;
- `R` — очистить набранные кадры;
- `Q` или `Esc` — выйти.
